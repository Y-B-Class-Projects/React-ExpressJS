About us

This site is an flower shop website example that includes user management.

The site has a flower database that only users who have registered on the site have access to.


Here are the types of users: 

  • Admin
  • Client
  • Employee
  • Supplier

An administrator will be able to update / delete / add users

An employee can only see client details and add a client.


Here are the details of users who have already registered for the site:

  • Username: admin Password: 123, Access level: admin
  • Username: client Password: 123, Level of access: client
  • Username: employee Password: 123, Access level: employee
  • Username: supplier Password: 123, Access level: supplier


The server side runs on ExpressJS, In addition, server-side and client-side uses 'Jquery' and 'AJAX' techniques.

The frontend of the client side is written with bootstrap.
